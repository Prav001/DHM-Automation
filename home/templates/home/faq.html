    What is DHM?

    DHM stands for Data Health Monitoring.

  • 2
    What is the use of this tool?

    This tool will be used to generate inconsistency reports, transaction statistics and month end reconciliation. The consistency report is used to check inconsistent data in the database. It uses a predefined set of consistency queries that compare the master table with other tables and finds the discrepancy in data. Transaction statistics is used to know about the frequency of the component of the aviation product, used by the costumer. .

    How to generate Inconsistency report?

    To generate an inconsistency report, click on the DHM tab from the side menu and select 'Generate Inconsistency Report'. Select the customer name, severity level, modules, bussiness components and run date and click on 'Genearte Report'.

    What if I want to add a customer?

    To add a customer to the app, you must have admin user rights. If you do, proceed to the Admin tab on the left, login with proper credentials and then go to the Masters tab on the left.
